# Notebooks Directory

This directory contains 10 sequential Jupyter notebooks for the anomaly detection project.

## Notebook Sequence

1. **01_data_exploration.ipynb** - Exploratory Data Analysis
2. **02_data_preparation.ipynb** - Data splitting and preprocessing
3. **03_domain_shift_generation.ipynb** - Generate shifted domain data
4. **04_patchcore_clean.ipynb** - Train PatchCore on clean domain
5. **05_padim_clean.ipynb** - Train PaDiM on clean domain
6. **06_evaluation_clean.ipynb** - Evaluate clean domain performance
7. **07_patchcore_shift.ipynb** - PatchCore domain shift experiments
8. **08_padim_shift.ipynb** - PaDiM domain shift experiments
9. **09_evaluation_shift.ipynb** - Evaluate shift domain performance
10. **10_results_visualization.ipynb** - Final results and visualization

Run them in order from 01 to 10.
